# 🎓 Student Scoreboard

A modern React + Vite based Student Scoreboard application used to manage student records, scores, and pass/fail status.

## 🚀 Features

- Add new students
- Enter student marks
- Automatic Pass / Fail result
- Stylish modern UI
- Responsive layout
- Built using React Components

## 🛠️ Technologies Used

- React.js
- Vite
- JavaScript
- CSS3

## 📂 Project Structure

src/
components/
Header.jsx
AddStudentForm.jsx
StudentTable.jsx
App.jsx
App.css

## 📊 Functionality

- Students with score **40 or above** = Pass ✅
- Students below **40** = Fail ❌
